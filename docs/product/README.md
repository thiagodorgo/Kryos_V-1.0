# Product Documentation

Documentação de produto do Kryos: visão, mapa de módulos, modelo de billing e a pesquisa de metodologia técnica.

## Conteúdo
- `product-vision.md`, `module-map.md`, `billing-model.md`, `customer-usage-control.md`, `internal-saas-control.md` — stubs de visão de produto (estágio structure-only).
- `estudo-doutoral-interfaces-10-saas.md` — estudo multi-caso das interfaces dos 10 principais SaaS de refrigeração, com método explícito, análise cognitiva (consciência situacional, fadiga de alarme ISA-18.2) e implicações de design para o Kryos.
- `metodologia-tecnica-10-saas-refrigeracao.md` — pesquisa e definição, validada em 10 SaaS de refrigeração do mundo, de: métodos de aquisição, estilos de visualização, objetos/funções/classes/esquema, metodologia de exibição, inteligência cognitiva e planos/cobrança. Ver também `../frontend/frontend-product-requirements.md` para os requisitos de UI derivados.

## Estágio
`structure-only`. Documentos de especificação, não implementação.
