# Kryos Web Dashboard

## Visão Geral
Interface de operação e gestão para usuários desktop. Desenvolvido em React e hospedado na EC2 junto ao Backend na fase 1.

## Foco
- Visualização de tendências e gráficos históricos.
- Gestão de configurações de usuários e permissões.
- Relatórios gerenciais.

## MVP Técnico (Fase 2)
Uma página HTML simples em `public/index.html` permite visualizar telemetria recente e enviar setpoints.
